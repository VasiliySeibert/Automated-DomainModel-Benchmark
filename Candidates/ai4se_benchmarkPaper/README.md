# ai4se_benchmarkPaper candidates

The `rule_based` candidate only — a re-implementation of
**Abdelnabi et al. (2020)** using a spaCy SVO + verb-lemma heuristic.
Self-authored by Vasiliy Seibert, 2024 (originally in
`ai4se_benchmarkPaper/benchmark/candidates/rule_based/utils.py`).

No LLM. Acts as a non-LLM baseline.

See [`rule_based/README.md`](rule_based/README.md) for the full
citation block and details on the algorithm.
